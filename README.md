# SBToNettyChat

## 简介
基于SpringBoot-WebSocket通用框架,结合Netty进行聊天社交，并记录聊天日志，异步存储

## 特点
异步处理，netty实时消息通信，JPA，WebSocket，数据库操作记录，聊天历史等

## 功能

    实时聊天
    异步CRUD处理消息日志
    获取聊天历史

## 配置

    application.yml 数据库配置
